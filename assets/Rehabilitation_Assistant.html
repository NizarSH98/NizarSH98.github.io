Rehabilitation Assistant

Machine learning model predicting patient motor engagement to inform therapists during recovery sessions.

  • Role: ML Engineer
  • Timeline: 2024
  • Tech: Python, scikit-learn, Pandas, Random Forest
  • Repository: GitHub search for Rehabilitation Assistant

Challenge

Clinicians needed proactive insights during physiotherapy sessions to ensure patients remained engaged without overexerting injured muscles.

Solution

I engineered a Random Forest classifier that ingests sensor readings from wearable IMUs and predicts engagement intensity in real time. Feature engineering pipelines clean noisy signals, compute rolling statistics, and standardize data for consistent inference.

  • Balanced class distribution using SMOTE to improve recall on under-represented engagement levels.
  • Exported model via ONNX for lightweight deployment on edge devices.
  • Therapist dashboard surfaced confidence intervals with friendly copy.

Results

Delivered 92% accuracy across test sessions and provided therapists with actionable prompts that shortened session planning by 20%.
