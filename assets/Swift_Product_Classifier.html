Swift Product Classifier

Rapid prototype that generates synthetic product data and trains a text classification pipeline within hours.

  • Role: ML Engineer
  • Timeline: 2023
  • Tech: Python, TensorFlow, Pandas, NumPy, FastAPI
  • Repository: GitHub search for Swift Product Classifier

Challenge

Build a proof-of-concept classifier for an e-commerce partner with limited labeled data and a tight deadline.

Solution

I created a bootstrapped dataset by programmatically generating realistic product descriptions, categories, and pricing metadata. TensorFlow powered a lightweight text classifier while FastAPI exposed inference endpoints for stakeholder review.

  • Automated preprocessing pipeline tokenized and vectorized descriptions with TF-IDF.
  • Experiment tracking scripts compared baseline models to neural architectures.
  • Evaluation dashboard summarized precision/recall for each category.

Outcome

The prototype achieved 89% macro F1 within 48 hours, enabling the client to greenlight a larger data collection effort.
