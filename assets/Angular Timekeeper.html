Angular Timekeeper

Lightweight Angular dashboard displaying live time, timezone awareness, and subtle UI animations.

  • Role: Front-End Developer
  • Timeline: 2022
  • Tech: Angular, TypeScript, SCSS, Animations
  • Repository: GitHub search for Angular Timekeeper

Challenge

Provide a visually delightful utility that keeps distributed teams aligned on local and remote time, while showcasing reusable Angular animation primitives.

Solution

The application renders synchronized clocks with subtle transitions driven by Angular’s animation API. Users can toggle between themes, switch locales, and pin favorite timezones. Service workers cache data to deliver instant launches.

  • Reactive state store driving clock updates and timezone conversions.
  • Accessible typography with high contrast for at-a-glance reading.
  • Custom animation utilities reused across button, card, and modal components.

Outcome

The Timekeeper module became a starter kit for future Angular experiments, reducing setup time for dashboard prototypes by 50%.
