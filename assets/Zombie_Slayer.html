Zombie Slayer

Custom C# action game demonstrating bespoke engine systems, AI behaviors, and responsive combat loops.

  • Role: Game Developer
  • Timeline: 2021
  • Tech: C#, MonoGame, Custom Engine, Pixel Art
  • Repository: GitHub search for Zombie Slayer

Challenge

Prototype a responsive combat experience with handcrafted enemy behaviors, level design, and soundtrack—without relying on heavyweight engines.

Solution

The game engine manages sprite batching, collision systems, and state transitions. AI-controlled zombies use finite state machines for patrol, chase, and attack phases. Particle systems add punch to hits and explosions.

  • Entity-component system isolates rendering, physics, and audio layers.
  • Input buffering ensures combo moves feel responsive.
  • Difficulty scalers adjust spawn rates based on player performance.

Outcome

The prototype evolved into a teaching tool for introducing students to game loop concepts and responsive input design.
