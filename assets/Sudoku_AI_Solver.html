Sudoku AI Solver

Interactive Sudoku board that visualizes a backtracking solver while teaching algorithmic thinking.

  • Role: Python Developer & Instructor
  • Timeline: 2022
  • Tech: Python, Pygame, Backtracking Algorithm
  • Repository: GitHub search for Sudoku AI Solver

Challenge

Create a teaching aid that demonstrates how constraint satisfaction algorithms explore the Sudoku search space, giving learners a visual sense of recursion and backtracking.

Solution

The solver renders a 9×9 grid and steps through the backtracking process in real time. Learners can pause, edit cells, and restart puzzles to see how the algorithm reacts. Performance optimizations cache valid candidates and prune branches quickly.

  • Color-coded hints show which numbers are being validated at each recursion level.
  • Keyboard shortcuts let players input their own puzzles and watch the solver respond.
  • Modular architecture splits puzzle validation, rendering, and event handling.

Results

The project became a core part of my introductory AI workshops, improving comprehension scores by 25% as measured through pre- and post-session quizzes.
