Angular User List

Responsive data browser supporting pagination, filtering, and instant search for large user datasets.

  • Role: Front-End Engineer
  • Timeline: 2023
  • Tech: Angular, RxJS, REST APIs, SCSS
  • Repository: GitHub search for Angular User List

Challenge

Back-office teams needed a performant interface to inspect thousands of customer profiles with complex filter criteria. The legacy solution suffered from long load times and lacked accessibility.

Solution

I architected a modular Angular application with lazy-loaded feature modules and reusable table components. RxJS streams coordinate API calls, debounced search, and optimistic updates.

  • Accessible table semantics with keyboard navigation and ARIA labels.
  • Smart pagination caching recent pages to reduce network chatter by 40%.
  • Chip-based filters allowing teams to bookmark saved queries.

Outcomes

Average query-to-render time dropped from 4.5s to under 1s, and support teams reported a 30% reduction in manual exports thanks to integrated CSV downloads.
